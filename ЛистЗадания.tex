{\parindent0pt
  4.1.2. Входные данные и требуемые результаты для программы:}

\begin{enumerate}
\item Входными данными для программной системы являются: данные
профиля пользователя в приложении; настройки пользователя; учетные
данные для авторизации; данные о совершенных пользователями заказах;
сведения о товарах, поступающие в режиме реального времени со стороны
сервера; данные о принятых фермерами заказах. 
\item Выходными данными для программной системы являются: страница
профиля пользователя; страница профиля продавца; информация о
фермерских хозяйствах; история заказов; статус текущих заказов; результаты
поиска и фильтрации; список продуктов с фотографиями, описанием, ценами,
отзывами; информация о хозяйстве, статистика продаж, управление
продуктами; информация о доставке.
\end{enumerate}
